> ESLint config for haifengfe


#### Getting Started

1. Install: `npm i eslint-config-haifengfe -D`.

2. Usage in your .eslintrc: 

    ```
    { "extends": "haifengfe" }
    ```

    **OR**, if you are using the .yml:
    ```yml
    extends:
      - 'haifengfe'
    ```

3. Use in Vue:
    ```
    { "extends": "haifengfe/vue" }
    ```

    **OR**, if you are using the .yml:
    ```yml
    extends:
      - 'haifengfe/vue'
    ```
